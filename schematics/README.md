# Festival Kit — Schematic Library

Place WorldEdit `.schem` or `.schematic` files in each directory.

| Directory          | Description                          |
|--------------------|--------------------------------------|
| `main_stage/`      | Main stage (largest, central)        |
| `techno_stage/`    | Techno / electronic stage            |
| `dnb_arena/`       | Drum & bass / bass music arena       |
| `acoustic_stage/`  | Smaller acoustic / chill stage       |
| `lighting_rig/`    | Lighting tower / truss structure     |
| `crowd_barrier/`   | Crowd control barrier segment        |
| `entrance_gate/`   | Festival entrance arch/gate          |
| `ferris_wheel/`    | Ferris wheel / carnival ride         |
| `camping_tent/`    | Camping tent plot                    |
| `spawn_platform/`  | Player spawn platform                |
| `path_block/`      | Path segment (connects zones)        |

## Usage

The `festivalworld build` pipeline auto-selects schematics
based on stage style and places them via WorldEdit commands.
